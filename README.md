# Finance Platform

A modern financial management platform with AI-powered insights.

## Features

- Transaction tracking and management
- Budget planning and tracking
- Monthly financial reports
- AI-powered financial insights
- Email notifications for budgets and reports

## Technologies Used

- Next.js
- Prisma
- PostgreSQL
- Clerk Authentication
- Inngest for background jobs
- Resend for email notifications

## Getting Started

### Prerequisites

- Node.js 18+
- PostgreSQL
- npm or yarn
